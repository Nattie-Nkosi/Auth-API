How to Run Auth API

First Time

  • Install NodeJS.
  • Install TypeScript globally using CLI using npm install -g typescript.
  • Check to see if TypeScript was installed successfully by running tsc -v.
  • Create a MongoURI key at MongoDB Atlas.
  • Create a .env file on root folder and place set your MongoURI key e.g (MONGO_URL='mongourl').
  • Inside a .env file set your JWT_KEY token (JWT_KEY='jwtKey').
  • Inside app.ts set up cors middleware, just set your client url URL_API to allow API to receive requests from your client.
  • On your CLI, Git Bash or Terminal run npm i to install dependencies.
  • Run the APi npm run dev.

Use Postman API Platform to send http requests on Auth APi

API ENDPOINTS

POST signup a user /api/users/signup
POST signin a user /api/users/signin
GET current user /api/users/currentuser
POST signout a user /api/users/signout
